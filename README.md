# CashFlow - Personal Expense Tracker

CashFlow is a simple personal finance web app built with Flask and
PostgreSQL. It helps users track expenses, record income, monitor
savings, and understand spending by category.

website link : https://cashflow-expense-tracker.vercel.app/
## Features

-   Secure login and registration
-   Track expenses and income
-   Automatic savings calculation
-   Category-wise spending breakdown
-   Export expenses to CSV
-   Private user-specific data

## Tech Stack

Python, Flask, PostgreSQL, HTML, CSS, JavaScript, bcrypt, Gunicorn

## Run Locally

``` bash
pip install -r requirements.txt
python app.py
```

Create a `.env` file with your `DATABASE_URL` and `SECRET_KEY` before
running the app.
